This mod makes creative flying faster by sprinting while flying in creative mode.

There are five config options in the fast_creative_flight.json file. <br>
"enableFastCreativeFlight":true, //This will enable or disable the mod. <br>
"allowOnServers":false, // If this is true this mod will work in multiplayer. <br>

These config options control the speed. Please change these to your preferred values: <br>
"speedBoostX":0.1, <br>
"speedBoostY":0.5, <br>
"speedBoostZ":0.1 <br>
